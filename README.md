# LivingWords LLM

**Open-source God-centered LLM** aligned with Christian theological doctrine.

Built for The Living Words Project to create biblically faithful AI tools for Bible studies, sermons, devotionals, and faith-based applications.

## Features
- Train small transformer models locally in Node.js
- CLI for training and generation
- Programmable API
- Server mode with chat interface (coming soon)
- Designed for theological alignment (integrates with CTTAF benchmark)

## Quick Start

```bash
npm install livingwords-llm
```

### CLI Usage
```bash
# Train on custom data
npx livingwords-llm train --data data/bible.txt --epochs 5

# Generate text
npx livingwords-llm generate "In the beginning God created"
```
